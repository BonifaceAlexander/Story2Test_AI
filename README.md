# Story2Test AI

A simple Streamlit app that takes Acceptance Criteria and generates **positive & negative test cases**, exportable as CSV and Excel.

## Run locally
```
pip install -r requirements.txt
streamlit run streamlit_app.py
```

## API Key
Enter your OpenAI API key in the sidebar (session only) or set:
```
export OPENAI_API_KEY="your-key"
```
